## 🚀 Quick Start

### 1. Get Bot Token from @BotFather
### 2. Get Your User ID from @userinfobot  
### 3. Setup ZarinPal account
### 4. Configure:

```python
# config.py
BOT_TOKEN = "your_token"
ADMIN_IDS = [your_id]
ZARINPAL_MERCHANT_ID = "your_merchant_id"
```

### 5. Install & Run:

```bash
pip install -r requirements.txt
python bot.py
```
